# Luxury CSS Website - Ushm Essence

A premium ecommerce landing page built using **HTML5 and CSS3** for **Ushm Essence**, designed to showcase luxury fragrances and gifting products with a modern user experience.

## Features
- Full-screen hero section
- Luxury dark theme UI
- Responsive product grid layout
- Hover animations and transitions
- Premium product showcase
- Mobile responsive design
- Brand-focused storytelling section

## Tech Stack
- HTML5
- CSS3

## Project Overview
This project was created to demonstrate advanced CSS styling skills by building a visually appealing ecommerce website. The website focuses on premium branding, smooth user experience, responsive layouts, and modern design aesthetics suitable for luxury ecommerce brands.

## Key CSS Concepts Used
- Flexbox
- CSS Grid
- Media Queries
- Hover Effects
- Box Shadows
- Gradients
- Responsive Design
